# Online haxe-formatter

### Dependencies

 * [Haxe](https://haxe.org/) 4.0.0-rc.2
 * [coconut.vdom](https://lib.haxe.org/p/coconut.vdom)
 * [formatter](https://lib.haxe.org/p/formatter)
 * [uglifyjs](https://lib.haxe.org/p/uglifyjs)

This project uses [lix.pm](https://github.com/lix-pm/lix.client) as Haxe package manager.
Run `npm install` to install the dependencies.

### Compile js

```
npm run haxe build.hxml
```
